## Install

```bash
npx create-react-app client
```

React `Router` and `Axios`

```bash
npm install react-router-dom axios
```

`Tailwind` install

```bash
npm install -D tailwindcss postcss autoprefixer
```

```bash
npx tailwindcss init
```

redux install

```bash
npm install @reduxjs/toolkit react-redux
```
